# 给小朋友做的算术练习

#### install

```
yarn install
yarn run
```


#### how to

快捷键          | 说明
--------------- | --
`r`             | restart
`esc`           | stop
`1` `J` `up`    | option1 
`2` `K` `right` | option2
`3` `L` `down`  | option3
`4` `;` `left`  | option4
`+` `pageup`    | level up
`-` `pagedown`  | level down
