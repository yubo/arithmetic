# 给小朋友做的算术练习

[demo](https://www.yubo.org/ss/index.html)

#### install

```
yarn install
yarn run
```


#### how to

快捷键         | 说明
-------------- | --
`r`            | restart
`esc`          | stop
`1` `J`        | option1 
`2` `K`        | option2
`3` `L`        | option3
`4` `;`        | option4
`+` `pageup`   | level up
`-` `pagedown` | level down
